Image Analyzer Guide

Comprehensive documentation for the Image Analyzer Web App.

Getting Started

The Image Analyzer is a powerful tool designed to visualize and compare biological image data (like ComDet Fiji plugin outputs). It runs directly in your browser but sends data to a secure backend for project storage.

Uploading Data

You can import data in CSV format. The application expects specific columns typical of Fiji ComDet results (e.g., 'Int', 'Area', 'Channel', 'Slice').

📂

Drag & Drop

Simply drag CSV files from your computer anywhere onto the page to upload them instantly.

🖱

File Button

Click the "Upload CSVs" button at the bottom of the sidebar to open the standard file picker.

Project Management

Save your work, including datasets, thresholds, and notes, to the server.

Saving Projects

Use the Save button in the sidebar.

  • Quick Save: If you've already saved the project, clicking "Save" overwrites the current file.
  • Save As: Click the small arrow next to Save to choose "Save As..." and create a new file.
  • Keyboard Shortcut: Press Ctrl + S (Windows) or Cmd + S (Mac).

Note: Unsaved changes are indicated by an amber glowing border on the Save button.

File Manager

The File Manager allows you to organize your projects on the server.

  • Navigation: Click folders to open them. Click ".." to go up.
  • Multi-Select: Hold Ctrl / Cmd to select multiple items.
  • Actions: Hover over an item to seeing Copy, Cut, and Delete buttons.
  • Clipboard: After copying/cutting, a "Paste" button appears in the toolbar.
  • Shortcuts: Cmd+C (Copy), Cmd+X (Cut), Cmd+V (Paste), Del (Delete).

Standard Visualization

Explore your data using various chart types and aggregation modes.

View Modes

  • Image: Group data by individual source image.
  • Well: Aggregate data by Well (expects filenames like 'A01_...').
  • Parameter: Aggregate by active parameter/condition.

Chart Types

  • Scatter: Plot individual data points (X=Index, Y=Value). Ideal for seeing distributions.
  • Histogram: View frequency distribution of values.
  • Box Plot: Statistical summary (Median, Quartiles). Hides legend for clarity.
  • Bar Chart: Compare aggregate values (Mean/Median).

Comparisons

Compare specific subsets of data.

  1. Click Enable Comparison in the sidebar.
  2. Select the datasets you want to compare from the list.
  3. Click Save Comp. to store this specific view.
  4. Access comparisons later from the "Saved Comparisons" list at the top.

Aggregation Controls

When in 'Well' or 'Parameter' view (and not in Histogram/Bar mode), aggregation controls appear. You can choose to plot All Points, or aggregate them to a single Mean or Median point per group.

FAQ

Does it work offline?

The visualization works offline, but Saving/Loading projects requires a connection to the hosting server.

What file formats are supported?

Currently, only CSV files with specific headers (Int, Area, etc.) are supported.
